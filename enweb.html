Encryption Algorithms

Caesar Cipher

A classic substitution cipher that shifts each letter by a fixed number down the alphabet. Simple yet effective for basic encryption needs.

Example: With a shift of 3, "HELLO" becomes "KHOOR".

Vigen√®re Cipher

A more secure polyalphabetic cipher that uses a keyword to determine variable shifts for each character, making frequency analysis attacks more difficult.

Example: With keyword "KEY", "HELLO" becomes "RIJVS".

Substitution Cipher

Each letter in the plaintext is replaced with another letter from a fixed mapping, creating a completely scrambled alphabet.

Example: A‚ÜíQ, B‚ÜíW, C‚ÜíE, etc., "ABC" becomes "QWE".

Morse Code

Not encryption in the traditional sense, but converts text to the dots and dashes of Morse code with customizable separators between characters.

Example: "SOS" becomes "... --- ...".

ROT13

A special case of the Caesar cipher with a fixed 13-character shift. Its simplicity makes it popular for basic text obfuscation.

Example: "HELLO" becomes "URYYB".
